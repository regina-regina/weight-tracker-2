# Weight Tracker

Приложение для отслеживания веса и прогресса похудения для iOS.

## Возможности

- Онбординг с настройкой целей и параметров
- Трекинг веса и измерений тела
- Расчет ИМТ, процента жира, TDEE
- Рекомендации по калориям
- Графики прогресса
- Прогнозы снижения веса для 3 темпов
- История всех записей
- Пастельный дружелюбный дизайн

## Технологии

- React Native + Expo
- Supabase (база данных + авторизация)
- React Navigation (навигация)
- react-native-chart-kit (графики)

## Установка

См. файл SETUP.md для подробных инструкций по настройке.

## Запуск

```bash
npx expo start
```

Отсканируй QR-код в приложении Expo Go на iPhone.
