# Иконки и изображения приложения

В `app.json` указаны пути к файлам, которые должны лежать в этой папке:

| Файл | Назначение |
|------|------------|
| **icon.png** | Иконка приложения (в меню устройства). Рекомендуемый размер: 1024×1024 px. |
| **adaptive-icon.png** | Передний план иконки для Android (без фона). Рекомендуемый размер: 1024×1024 px. |
| **splash-icon.png** | Картинка на экране загрузки (splash). Обычно логотип или иконка. |
| **favicon.png** | Иконка для веб-версии. Обычно 48×48 или 32×32 px. |

Сейчас в папке есть только иконки для нижнего меню: `icon-home.png`, `icon-history.png`, `icon-chart.png`, `icon-profile.png`.

**Что сделать перед сборкой под прод:**  
Добавь в `assets` недостающие файлы с именами выше. Можно временно скопировать одну из существующих иконок (например, `icon-home.png` → `icon.png`), чтобы сборка не падала, а потом заменить на финальные иконки и splash.
