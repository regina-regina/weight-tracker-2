# Как установить приложение для теста

Есть два способа — через браузер (PWA) и через TestFlight (iPhone).

---

## Вариант 1: Установка через браузер (PWA)

Подходит для теста на телефоне или компьютере без App Store.

1. Открой в браузере **Safari** (на iPhone) или **Chrome** (на Android/компьютере) ссылку:
   https://regina-regina.github.io/weight-tracker-2/

2. Войди в приложение (регистрация или вход по email).

3. Чтобы добавить иконку на главный экран:
   - **iPhone (Safari):** нажми «Поделиться» → «На экран „Домой“» → «Добавить».
   - **Android (Chrome):** меню (три точки) → «Установить приложение» или «Добавить на главный экран».

4. Дальше можно открывать приложение с главного экрана как обычную программу.

---

## Вариант 2: Установка через TestFlight (только iPhone)

Если тебе прислали приглашение в TestFlight:

1. Установи из App Store приложение **TestFlight** (бесплатно).
2. Открой письмо с приглашением и нажми ссылку «Открыть в TestFlight».
3. В TestFlight нажми «Установить» — приложение появится на главном экране.
4. Запускай с главного экрана. Обновления приходят через TestFlight.

Если приглашение не пришло — напиши тому, кто добавляет тестеров: нужно проверить email и переслать приглашение.
